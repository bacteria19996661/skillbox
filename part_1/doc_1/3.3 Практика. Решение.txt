# Задача 1. Фруктовый сад
#
# Маша проводит исследования корреляции размеров фруктов в зависимости от их расположения на участке.
# Она взяла яблоко, грушу и персик диаметрами 8, 5 и 3 сантиметра соответственно и вычислила коэффициент зависимости по своей формуле.
#
# Реализуйте программу, которая вычисляет значение выражения
#
#
#
#
#  где a, b и c — переменные, в которых лежат числа 8, 5 и 3 соответственно. Выведите результат на экран.
#
#
#

a, b, c = 8, 5, 3
result = ((a / (b + c)) - a) / c
print(result)

# Таким образом при помощи скобок мы устаналиваем свой порядок выполнения операций:
# 1) (b + c)  -- выполнится первым делом
# 2) (a / (b + c))  -- затем "а" будет поделено на результат вычисления первой операции
# 3) ((a / (b + c)) - a)  -- от результата вычислений №2 далее будет отниматься переменная "а"
# 4) ((a / (b + c)) - a) / c  -- и в конце результат вычислений будет поделен на "c"
